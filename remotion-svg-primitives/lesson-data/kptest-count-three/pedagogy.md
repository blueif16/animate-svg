# kptest-count-three — pedagogy

lesson kind: math-insight

reinforcement plan (stated once): the count-climb cue carries its own practice — three one-to-one arrivals inside one cue (one count word + one tag per apple, paced with breath gaps), and that tri-repetition IS the age-appropriate drill for the 一/二/三 sequence and one-to-one (the child half-knows the sequence; no separately-templated repeat cue is earned). Cardinality is an insight: one clean delivery whose picture shows 3→whole, with the total named aloud as the cardinal word the child is meant to produce — that naming is the teaching, not a separate recap. No recap cue beyond the cardinality beat itself (it doubles as the landing/retrieval). Two cues; the ~25s emerges from three paced arrivals + one cardinality landing, never from padding.

## cue-1 — count-climb
```yaml
discovery: each apple gets exactly one count word, in order (一, 二, 三) — one object = one count, counted up (one-to-one correspondence + the counting sequence, which the child half-knows)
stage: concrete (the identical apples are the objects counted) with represent tags (the running numeral 1, 2, 3 attaches to each apple as its count lands; the apples stay identical and each keeps its tag once placed — the one-to-one is stable and reusable)
focal: the arriving apple and the number tag that attaches to it on its count word; apples already counted stay put and stay quiet (no shuffle, no re-entry) — continuity = each apple keeps its identity and its count
reinforcement: built-in tri-repetition inside the cue (three paced arrivals, one count word + one tag each) is the drill for this sequence the child half-knows; no extra repeat move is earned
```

## cue-2 — cardinality
```yaml
discovery: the last number said (三 / 3) tells how many there are altogether — the 3 that labeled the third apple now stands for the whole group (cardinality)
stage: represent (the numeral 3 stops being one apple's tag and becomes the set's number — the cardinal bridge; no equation, no numeral beyond 3)
focal: the single 3 relocating / re-scoping to stand for all three apples at once, so its role changes from per-object tag to whole-group number; the three apples stay quiet underneath it. The picture delivers the 3→whole relation FIRST; only then does the voice name the total (三个 / 一共有三个) — naming the cardinal word after the picture has shown it is the teaching, not leakage
reinforcement: none beyond the one delivery — an insight that clicks once; no recap cue, the cardinality beat itself is the landing
```
