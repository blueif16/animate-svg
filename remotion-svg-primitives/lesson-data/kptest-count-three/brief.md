# kptest-count-three

**Audience.** Ages 3–5, Chinese (Mandarin) medium.
**Length.** ~25s — HINT for scope, not a target the composer pads to.
**Builds on.** none
**Style.** default

## Knowledge point
The child leaves able to count three identical objects one-by-one (一, 二, 三) and understands that the last number said names how many there are altogether (cardinality).

## The one beat
Three identical apples appear one at a time; as each lands, a number tag (1, then 2, then 3) attaches to it; at the end a single "3" stands for the whole group.

## Out of scope
No addition, no numerals beyond 3, no comparing quantities. Just count-and-total to three.

## Continuity
Reusable counting primitives (countable objects + number tags). Each apple is identical and keeps its identity once placed.

## Narration notes
Short Mandarin counting; avoid 3-item run-on comma lists (use 和 / breath groups) per Aoede TTS calibration.
