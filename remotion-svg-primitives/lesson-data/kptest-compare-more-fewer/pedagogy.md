# pedagogy — kptest-compare-more-fewer

lesson kind: math-insight (the paired picture reveals MORE/FEWER) WITH an L2-style acquisition layer — the two 比-comparison utterances 五比三多 / 三比五少 are acquisition targets the child hears + copies, so the VOICE names them in full (§4 carve-out) and they earn spaced reinforcement + wait-time; the MATCHING insight itself is picture-delivered and the voice does NOT pre-name which group has more.

Acquisition targets (drilled, voiced, retrieved): **五比三多** and **三比五少** — one per direction. Insight (picture-delivered once, recalled later): one-to-one matching decides MORE/FEWER, and the same paired picture says both directions. Whole-set sanity check: the two phrases are co-equal (§8 breadth) — each gets its own clean delivery cue + a wait-time gap + a slot in the recap; the bidirectional-same-picture insight is the keystone and gets the longest dwell.

---

```
cue: intro
discovery:     none yet — announces the topic (谁多谁少 / comparing two groups) so the child knows what this video is for.
stage:         represent
focal:         the lesson-intro-card title appearing (intro card per CLAUDE.md; not a teaching beat)
reinforcement: none (framing, one pass)
```

```
cue: two-groups
discovery:     here are two groups of dots — five red on top, three blue below — laid in two aligned rows; nothing is compared yet, the child just takes in "two groups, different amounts."
stage:         concrete
focal:         the two dot rows arriving (5 red top, 3 blue below), identical dots so only COUNT differs — color is group ID, not size
reinforcement: none (setup, one pass) — voice names "two groups" and the amounts; the comparison is withheld for the matching beat
```

```
cue: match
discovery:     when each top dot drops a line straight down to one bottom dot, two top dots are left with NO partner — the child SEES the surplus; the leftover pair is the proof, before any word for it.
stage:         represent
focal:         the pair-connector lines growing top→bottom one-to-one, ending with the two unpaired top dots marked as having nobody (unmatched-slot ghost)
reinforcement: none (insight, one clean delivery) — §4: voice names the ACTION ("each top dot finds a partner"), the picture DELIVERS the two left over; do NOT pre-say "five is more"
```

```
cue: more-direction
discovery:     the two leftover dots have a name: 五比三多 — five is more than three. The acquisition target is the WHOLE utterance (names both groups + direction), bound to the surplus the child just saw.
stage:         symbolize
focal:         the surplus pair pulsing as the voice says 五比三多; a 多-direction comparison-symbol (>) attaching top-over-bottom
reinforcement: ACQUISITION — model 五比三多 slowly on its instance, then a choral invite ("跟我说：五比三多") + a learner-response gap (≥3–5s held silence) before moving on; replays once on the same surplus picture
```

```
cue: fewer-direction
discovery:     THE SAME paired picture, read the other way, is 三比五少 — three is fewer than five. Nothing in the picture changes; only the direction of the reading flips. This is the keystone: 多 and 少 are two names for ONE comparison.
stage:         symbolize
focal:         the identical pairing held (no re-layout — same dots, same lines), the reading-focus sliding from the surplus (多) to the short row (少) as the voice says 三比五少
reinforcement: ACQUISITION + keystone insight — longest dwell; model 三比五少 slowly, choral invite + learner-response gap (≥3–5s); explicitly hold the unchanged picture so the child feels "same picture, two ways to say it"
```

```
cue: not-by-size
discovery:     which group has MORE is decided by matching, NOT by how spread out the dots are — spreading the three blue dots wide so the row looks longer does not make them more; re-matching still leaves the five with two over.
stage:         represent
focal:         the three blue dots spreading wider (row looks longer) then the lines re-pairing — still two red left over; the surplus is invariant under spreading
reinforcement: insight guard, one pass — §4: voice asks the question ("看起来变多了吗？"), the picture DELIVERS "no, still two left over"; do not pre-answer
```

```
cue: recap
discovery:     retrieval — both utterances come back on the original picture: 五比三多 AND 三比五少 name the same paired groups; the child re-produces both, closing the loop.
stage:         symbolize
focal:         the recap-spotlight walking back the surplus → 五比三多, then the short row → 三比五少, each as a complete utterance
reinforcement: SPACED RECALL of both co-equal targets (§8 breadth) — each phrase retrieved as a full utterance with its own brief say-it gap; equal airtime to 多 and 少, neither name-checked
```
