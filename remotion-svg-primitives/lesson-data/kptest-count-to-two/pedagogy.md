# pedagogy — kptest-count-to-two

lesson kind: math-insight
(Discovery is the count-then-total math idea, not an L2 sound to acquire. Mandarin counting words 一/二 for ages 3–5 are near-known; the narration may voice them (§7 count-match) but they are not acquisition targets, so this is NOT a §9 language lesson — no choral/echo/wait-time floors apply. Reinforcement is light and insight-shaped per §8.)

## Cues

```
cue-id: cue-1-count
discovery:     The child discovers the ACTION of counting — two identical apples arrive one at a time, and each gets the next number word in turn (一, then 二), so counting means tagging each object with the next number.
stage:         concrete
focal:         Each apple as it lands + its ordinal tag attaching (1, then 2). Both apples identical so only the count sequence differentiates them.
reinforcement: none (insight, one pass) — counting-to-two is low-novelty for 3–5; one clean delivery is enough; cue-2's total re-voices the count and absorbs the recall role.
```

(§7 — spoken count MUST match on-screen tags: narrator says 一 as tag 1 attaches to apple 1, 二 as tag 2 attaches to apple 2. Tags appear IN TURN, never both at once, so the child does the count action alongside the picture.)

```
cue-id: cue-2-cardinality
discovery:     The child discovers CARDINALITY — the last number you say when counting (二 / "2") names how many there are ALTOGETHER, not just the second one; one "2" stands for the whole group of two.
stage:         symbolize
focal:         The two per-object tags (1, 2) consolidating into a single group "2" that represents the total. This consolidation/move IS the teaching; apples stay quiet.
reinforcement: light insight — cardinality is the genuinely new idea, so it gets ONE clean delivery (tags → one 2) plus a minimal spaced recall: re-voice the total "二个" naming the whole at the final hold (the recap punctuation, doubling as §6's one reserved pulse on the central teaching object).
```

## Reinforcement plan (stated once)

Two cues, insight-shaped (§8), paced so length EMERGES from need (~20s is a floor-emergent hint, not a target):
- cue-1 (count action): single pass — the act of tagging in turn IS the model; no replay.
- cue-2 (cardinality): one delivery + one end recall (re-voiced total). This is the lesson's only repeat and its reserved pulse (§6).
No choral/echo/wait-time: not an acquisition target (§9 carve-out does not apply). The apples are identical and keep identity once placed (continuity from brief) — identity-stability is what lets each ordinal tag unambiguously land, and is NOT itself a teaching element.
No setup beat: counting-to-two has no labor/cost that needs making visible (§1 fourth shape not warranted); start at the new-content beat.
