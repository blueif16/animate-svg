# kptest-count-to-two

**Audience.** Ages 3–5, Chinese (Mandarin) medium.
**Length.** ~20s — HINT for scope, not a target the composer pads to.
**Builds on.** none
**Style.** default

## Knowledge point
The child leaves able to count two identical objects one-by-one (一, 二) and understands that the last number said names how many there are altogether (cardinality).

## The one beat
Two identical apples appear one at a time; as each lands, a number tag (1, then 2) attaches to it; at the end a single "2" stands for the whole group.

## Out of scope
No addition, no numerals beyond 2, no comparing quantities. Just count-and-total to two.

## Continuity
Reusable counting primitives (countable objects + number tags). Each apple is identical and keeps its identity once placed.

## Narration notes
Short Mandarin counting; avoid run-on comma lists (use 和 / breath groups) per Aoede TTS calibration.
