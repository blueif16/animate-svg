# kptest-first-second-third — pedagogy (Wave 0)

lesson kind: math-insight (position→ordinal mapping: count-from-front, find-by-word) + light L1 word acquisition (第一 / 第二 / 第三). §4 acquisition carve-out applies to the three ordinal WORDS — the voice names each fully, always in a complete natural utterance ("小动物排第一", never a bare "第一。"). The inverse mapping in the retrieval cues is INSIGHT — the question prepares, held silence, the PICTURE answers; narration never pre-names who.

Global constraints serving the discoveries:

- The FRONT of the queue must be visually unambiguous in every cue (one fixed front marker; every count sweep originates there). "Counted from the front" IS the rule being taught — an ambiguous front kills every discovery below.
- Ordinal labels are the Chinese words 第一/第二/第三 as voiced read-along chips attached to animals, never digits (brief: no written digits as teaching objects).
- Specific animal species are downstream's choice (storyboard + catalog); pedagogy refers to animal 1/2/3 by queue position.

Reinforcement plan (stated ONCE; per-cue lines reference it):

| target | exposures (spaced, interleaved) | count |
|---|---|---|
| 第一 | model c2 · recount c3 · recount c4 · choral c7 | 4 |
| 第二 | model c3 · recount c4 · retrieval c5 (question + confirm) · choral c7 | 4–5 |
| 第三 | model c4 · retrieval c6 (question + confirm) · choral c7 | 3–4 |
| rule: count from the front | delivered c3 · generalized c4 · retrieved c5, c6 · recapped c7 | 5 |

- Targets are half-known L1 words (low-mid novelty) → ~4 spaced exposures each is the honest low end of the §8 floor; balanced across the three co-equal words (no over-drilling one).
- Two learner-response gaps (typed `gap`, ≥3–5s held silence, line static) after the questions in c5 and c6 — the child answers before the picture reveals.
- Recount-on-arrival (c3, c4) doubles as built-in spaced recall; no word is drilled in a block — interleaving is structural.
- Emergent length ≈ 55–65s, consistent with the brief's hint without padding.

## cue: intro

```
discovery:     A line poses an order question — "谁排第几?" — that the child can't yet answer precisely; positions in a line have their own names. (felt limit, §1 fourth shape; motivates c2–c4 and returns as the c5/c6 questions)
stage:         represent
focal:         topic card text + the spoken question appearing; nothing else moves.
reinforcement: none (setup).
```

## cue: line-first

```
discovery:     The position nearest the front marker has a name: the animal standing there 排第一. (new unit: first ordinal name bound to the front position)
stage:         concrete
focal:         animal 1 walks in and settles at the front marker; the 第一 chip attaches AFTER it settles, on the voiced complete utterance.
reinforcement: model exposure #1 of 第一 (see plan).
```

## cue: line-second

```
discovery:     The next position behind earns its name by COUNTING FROM THE FRONT — 第一、第二 — so the new arrival 排第二. (new operation: count positions from the front)
stage:         concrete
focal:         animal 2 settles behind animal 1; a count sweep starts at the front marker — 第一 (chip pulses, recount) then 第二 (new chip attaches); spoken count matches the sweep (§7).
reinforcement: model 第二 + spaced recount of 第一 (see plan).
```

## cue: line-third

```
discovery:     The rule holds for every new arrival: recount from the front and the newest one 排第三. (new unit 第三 via the same operation — the brief's one beat completes)
stage:         concrete
focal:         animal 3 settles; full front-to-back count sweep — 第一、第二、第三 — each chip pulsing exactly on its spoken word (§7), 第三 attaching last.
reinforcement: model 第三 + spaced recounts of 第一 and 第二 (see plan).
```

## cue: ask-second

```
discovery:     The mapping runs in reverse: given the WORD 第二, the child can find WHO by counting from the front themselves. (new operation: word → position — the KP's target competence)
stage:         concrete
focal:         question "谁排第二?" voiced → held silence (typed learner-response gap ≥3–5s, line static) → animal 2 steps forward, its 第二 chip pulses, voice confirms with the complete utterance. Picture delivers the answer (§4).
reinforcement: retrieval of 第二 + the rule; first learner-response gap (see plan).
```

## cue: ask-third

```
discovery:     The same find-by-word works for any of the three words — here 第三. (consolidation of the inverse operation)
stage:         concrete
focal:         question "谁排第三?" voiced → held silence (typed learner-response gap ≥3–5s) → animal 3 steps forward, 第三 chip pulses, voice confirms with the complete utterance.
reinforcement: retrieval of 第三 + the rule; second learner-response gap (see plan).
```

## cue: recap-count

```
discovery:     none new — integration: 第一、第二、第三 is a count you can say from the front of any line (generalizes beyond this queue; closes all three targets).
stage:         concrete
focal:         choral invitation ("跟我一起，从前面数……") then one slow front-to-back sweep; each chip pulses exactly on its spoken word (§7); nothing else moves.
reinforcement: choral spaced recall of all three targets + the rule — final exposures per plan.
```
