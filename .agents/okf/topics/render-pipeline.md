# Why / how it works (the lifecycle, end to end)
`w5-render` is the final wave — `render-complete-lesson.mjs`'s `main()` orchestrates the full pipeline (optional
voice regen, lint, bundle+render, then a deterministic 2-pass ffmpeg `loudnorm`: pass 1 measures the master mix,
pass 2 applies the measured params) into `out/<lessonId>/`. `lesson-measured.mjs` is the companion MEASURED pass
that closes the declared⟺measured bijection from [[composer-scene-assembly]]'s manifest contract — it reads
each declared element's bbox back off the actual render (never mirrors geometry in the manifest) and separately
asserts the render's loudness lands within `LUFS_TARGET ± LUFS_TOLERANCE` of the loudnorm target.
