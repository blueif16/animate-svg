# Why / how it works (the lifecycle, end to end)

even include `src/`); the actual sketch-overlay CODE is the `TeacherMark` primitive the composer (`w4a`)
instantiates per w4b's spec. `TeacherMark` renders a hand-drawn ink mark (underline/circle/arrow/check, per
`TeacherMarkKind`) anchored to a scene element with a `BoilConfig` "hand-wobble" animation and a `SettleConfig`
fade, driven by `useCurrentFrame` only (no `Date.now`/`Math.random` — deterministic per the lesson-build's
render-determinism rule).
